# 호그와트 교수봇

호그와트 마법학교의 학사 관리를 담당하는 마스토돈 봇입니다. 학생들의 출석과 과제 제출을 관리하고, 기숙사 점수 시스템을 운영합니다.

## 주요 기능

### 학사 관리
- **신입생 등록**: 새로운 학생의 입학 처리
- **출석 관리**: 일일 출석 체크 및 보상 지급
- **과제 관리**: 과제 제출 확인 및 보상 지급
- **기숙사 시스템**: 기숙사별 점수 관리

### 자동화 기능
- 매일 오전 9시 출석 시작 알림
- 매일 오후 10시 출석 마감
- 출석/과제 제출 시 자동 갈레온 및 기숙사 점수 지급
- 겨울 세계관에 맞는 날씨 정보 제공

### 보상 시스템
- **출석**: 2갈레온 + 1기숙사점수
- **과제**: 5갈레온 + 3기숙사점수 (교수 태그 필수)
- 각각 일일 1회 제한

## 설치 및 설정

### 1. 저장소 클론
```bash
git clone <repository-url>
cd hogwarts-professor-bot
```

### 2. 의존성 설치
```bash
bundle install
```

### 3. 환경변수 설정
`.env` 파일을 생성하고 다음 내용을 설정하세요:

```env
# 마스토돈 설정
MASTODON_BASE_URL=https://your-mastodon-instance.com
MASTODON_TOKEN=your_access_token_here

# 구글 시트 설정
GOOGLE_CREDENTIALS_PATH=path/to/service-account-key.json
GOOGLE_SHEET_ID=your_google_sheet_id_here
```

### 4. 구글 시트 설정

#### 4.1 Google Cloud Console 설정
1. [Google Cloud Console](https://console.cloud.google.com/)에서 새 프로젝트 생성
2. Google Sheets API 및 Google Drive API 활성화
3. 서비스 계정 생성 및 JSON 키 파일 다운로드
4. 다운로드한 JSON 파일을 `GOOGLE_CREDENTIALS_PATH`에 지정한 경로에 저장

#### 4.2 구글 시트 생성 및 권한 설정
1. 새 구글 스프레드시트 생성
2. 서비스 계정 이메일에 편집 권한 부여
3. 시트 URL에서 ID 추출하여 `GOOGLE_SHEET_ID`에 설정

## 구글시트 구조

### 1. '사용자' 탭
| 열 | 헤더명 | 설명 |
|---|---|---|
| A | ID | 마스토돈 계정명 |
| B | 유저명 | 학생 이름 |
| C | 갈레온 | 보유 갈레온 |
| D | 소지품 | 아이템 목록 |
| E | 비고 | 메모 |
| F | 기숙사 | 소속 기숙사 |
| G | 마지막출석일 | 최근 출석일 (YYYY-MM-DD) |
| H | 마지막과제일 | 최근 과제 제출일 (YYYY-MM-DD) |
| I | 마지막베팅일 | 베팅 기록 (상점봇 호환용) |

### 2. '응답' 탭
| 열 | 헤더명 | 설명 |
|---|---|---|
| A | ON/OFF | 응답 활성화 여부 |
| B | 인식 키워드 | 명령어 키워드 |
| C | 출석 툿 내용 | 툿 내용 (사용안함) |
| D | 답변 출력 | 실제 응답 메시지 (랜덤 출력) |

### 3. '기숙사점수' 탭
| 열 | 헤더명 | 설명 |
|---|---|---|
| A | 기숙사 | 기숙사명 |
| B | 총점 | 누적 점수 |
| C | 최근업데이트 | 마지막 점수 변경 기록 |

## 사용법

### 학생 명령어

#### 입학 등록
```
[입학/홍길동]
```
- 새로운 학생으로 등록
- 20갈레온 초기 지급

#### 출석 체크
```
[출석]
또는
출석
```
- 출석 시간: 오전 9시 ~ 오후 10시
- 보상: 2갈레온 + 1기숙사점수
- 일일 1회 제한

#### 과제 제출
```
[과제] @교수님 오늘 과제 제출합니다
```
- 과제 제출 시간: 오전 9시 ~ 오후 10시
- 교수 태그 필수
- 보상: 5갈레온 + 3기숙사점수
- 일일 1회 제한

## 실행

### 봇 시작
```bash
ruby main.rb
```

### 연결 테스트
봇 시작 시 자동으로 다음 항목들을 확인합니다:
- 마스토돈 서버 연결
- 구글 시트 연결
- 필요한 워크시트 존재 여부

## 기숙사 시스템

### 기숙사 목록
- 그리핀도르
- 슬리데린  
- 레번클로
- 후플푸프

### 점수 획득 방법
- 출석: +1점
- 과제 제출: +3점

## 날씨 시스템

겨울 세계관에 맞는 날씨 정보를 제공합니다:
- 맑은 겨울, 눈보라, 흐린 겨울, 서리, 안개 낀 겨울, 강풍을 동반한 겨울, 진눈깨비
- 온도: -18°C ~ 5°C 범위
- 각 날씨별 적절한 조언 제공

## 문제해결

### 일반적인 문제들

#### 마스토돈 연결 실패
- `MASTODON_BASE_URL`과 `MASTODON_TOKEN` 확인
- 토큰 권한 설정 확인 (read, write, push)

#### 구글 시트 연결 실패
- 서비스 계정 JSON 파일 경로 확인
- 시트 공유 권한 확인 (서비스 계정 이메일에 편집 권한)
- Google Sheets API 활성화 확인

#### 출석/과제가 인식되지 않음
- 정확한 명령어 형식 사용 확인: `[출석]`, `[과제]`
- 시간 제한 확인 (9:00-22:00)
- 과제 제출 시 교수 태그 확인

#### 응답이 없거나 이상함
- '응답' 탭의 ON/OFF 설정 확인
- 인식 키워드와 답변 출력 내용 확인

### 로그 확인
봇 실행 시 콘솔에서 다음 정보를 확인할 수 있습니다:
- 멘션 수신 로그
- 명령어 처리 결과
- 오류 메시지
- 연결 상태

## 주의사항

1. **일일 제한**: 출석과 과제는 각각 하루에 1번만 가능합니다.
2. **시간 제한**: 출석과 과제는 오전 9시부터 오후 10시까지만 가능합니다.
3. **교수 태그**: 과제 제출 시 반드시 교수를 태그해야 합니다.
4. **구글 시트**: 수동으로 시트를 수정할 때는 형식을 준수해주세요.
