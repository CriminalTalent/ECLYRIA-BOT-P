# 🧙‍♂️ 호그와트 교수봇 - ECLYRIA-BOT-P

마스토돈 기반으로 작동하는 **호그와트 교수봇**입니다.  
학생들의 출석과 과제 제출을 확인하고, 갈레온과 기숙사 점수를 지급하며, Google 스프레드시트를 통해 학적부를 관리합니다.

---

## 🎓 주요 기능

| 기능 | 설명 |
|------|------|
| 📡 멘션 명령어 감지 | [출석], [과제], [입학/이름], [점수부여], [점수차감] 등의 명령어 자동 처리 |
| 📊 Google Sheets 연동 | 사용자 정보 및 기숙사 점수를 실시간으로 저장 및 조회 |
| 🕒 자동화 시스템 | 09:00 출석 시작 안내, 22:00 출석 마감 공지 |
| 💬 교수님 말투 | 모든 응답은 따뜻하고 지적인 말투로 구성 |
| 🧮 보상 시스템 | 갈레온과 기숙사 점수 자동 지급 |
| 🔒 출석/과제 1일 1회 제한 | 시간대 기반 처리 및 중복 방지 |
